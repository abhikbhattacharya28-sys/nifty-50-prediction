Model Architecture v2 EXP-WEIGHTED

8 loopholes identified in v1 and patched in v2. All fixes are live and feeding into today's prediction.

⚠ Accuracy figures are walk-forward out-of-sample — no data leakage
✓
Fix 1 — Exp. Weighted Retraining
v1 weighted all 250+ days equally. v2 uses exponential decay (half-life=60d) so recent days carry more weight, adapting faster to market regime changes.
✓
Fix 2 — Walk-Forward OOS Accuracy
v1 reported in-sample (training data) accuracy — an inflated number. v2 uses strict walk-forward out-of-sample evaluation with no data leakage.
✓
Fix 3 — BankNIFTY De-Duplicated
v1 used NSEI as proxy for BankNIFTY — feeding the same data twice. v2 uses real ^NSEBANK data; if unavailable, the feature is dropped entirely (not faked).
✓
Fix 4 — Confidence Percentile-Based
v1 confidence was arbitrary (|pred|/1.5). v2 computes confidence as the percentile of the prediction magnitude vs the full historical distribution.
✓
Fix 5 — India VIX Added
India VIX (^INDIAVIX) — the fear gauge — was missing in v1. Now included as two features: prev-day change and absolute level. High VIX is bearish for NIFTY.
✓
Fix 6 — Auto-Tuned Ridge Alpha
v1 used a fixed alpha=1.0. v2 tests 8 alpha values via TimeSeriesSplit CV and selects the best each day. Current optimal alpha: 10.0
✓
Fix 7 — 4 New Technical Features
Added: 200-day SMA distance (long-term mean reversion), 10-day Rate of Change (medium momentum), ATR% (volatility regime), India VIX level. Features grew from 16 to 20.
✓
Fix 8 — Magnitude Accuracy Tracked
v1 only tracked directional accuracy. v2 also tracks magnitude accuracy: how often the predicted % move lands within 0.25%, 0.50%, and 1.00% of the actual move.
